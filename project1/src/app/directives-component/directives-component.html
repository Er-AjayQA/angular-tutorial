What is directives?

  • Directive is a class that adds additional behaviours to the elements in your application.
  • A feature in angular that help you to provide more power to DOM elements.
  • Examples:
    • if-else condition
    • For loop
    • Add style
    • ngIf
    • ngFor
    • ngStyle
    • ngSwitch
    • ngClass

Type of directives?

  • Component Directives
    • The most common type of directives.
    • Used in component template files.
  • Structural Directives
    • Change the structure of the DOM by adding, removing or manipulating elements.
  • Attribute Directives
    • Modify the appearance or behaviour of an existing element.
